## üêõ **Common Issues & Solutions**

### Issue: "AI service not configured"
**Solution**: Add `GROK_API_KEY` to `Backend/.env` and restart backend

### Issue: "Authentication required"
**Solution**: Log in to the Cloud IDE

### Issue: Generic/Template Responses
**Solution**: Check that:
- Grok API key is valid
- Backend is using the new controller
- Frontend is calling `/api/ai/*` endpoints

### Issue: Code Generation Fails
**Solution**: 
- Check Grok API quota
- Verify network connection
- Check backend console for errors
